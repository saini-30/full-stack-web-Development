Frames

Use <frameset> and <frame> tags for dividing content into frames:

<frameset cols="50%,50%">
    <frame src="page1.html">
    <frame src="page2.html">
</frameset>

DHTML & CSS

CSS can be added in three ways:

  • Inline: style="color: red;"
  • Internal: <style> tag in <head>
  • External: Link to a CSS file
